## 🎨 DICAS DE IMPLEMENTAÇÃO

### Facebook Ads Manager
```
1. Ir para: facebook.com/ads/manager
2. "Criar campanha"
3. Objetivo: Tráfego ou Conversões
4. Audiência: Usar segmentos definidos
5. Orçamento: Conforme tabela acima
6. Duração: 3 dias (teste) ou 7 dias (escala)
7. Agendamento: Agendar por horário
```

### Instagram (Organic)
```
1. Postar nos horários recomendados
2. Use hashtags com 20-30 tags
3. Responda aos comentários em <1h
4. Use Stories para drive ao post
5. Cross-promote com Facebook
```

### Rastreamento
```
UTM Params: utm_source=facebook&utm_medium=ads&utm_campaign=kaza_awareness
Pixel: Facebook Pixel ID registrado
Analytics: Google Analytics + MixPanel
Conversão: Trial signup rastreado
```

## 🚨 SINAIS DE ALERTA

Se isso acontecer, ajuste rapidamente:

| Sinal | Ação |
|-------|------|
| CPC > R$2 | Pausar, revisar copy |
| CTR < 0.5% | Trocar imagem/vídeo |
| CAC > R$10 | Ajustar targeting |
| Trial rate < 80% | Issue com onboarding |
| Churn > 10% | Problema com UX |
